# AGENTS.md

## 概要

- 言語: Go
- 成果物: シングルバイナリ CLI (`catr`)

## 基本方針

- 変更は最小限に留める
- 可読性を優先する
- 影響範囲を明示する

## 実装ルール

- 関数は原則 35 行以内
- if はガード節優先、ネスト最大 2 段
- 変数名・関数名は簡潔にする
- KISS / DRY / リーダブルコードを守る

## 確認コマンド

```bash
gofmt -w .
go vet ./...
go test ./...
```

## レビュー方法

- self-review を実施する
- 追加レビューは `ts-cli-review` 相当観点で手動レビューする
